# AI Code Refactorer & Complexity Analyzer

A web-based tool that uses the **Gemini 2.5 Flash API** to automatically refactor code for efficiency and provide a detailed analysis of Time and Space complexity for every method.

## 🚀 Features
- **Smart Refactoring:** Clean up messy code into professional, efficient snippets.
- **Complexity Breakdown:** Get O(n) Time and Space complexity cards for every function.
- **Multi-Language Support:** Works with Java, Python, JavaScript, and C++.
- **Dark Mode Support:** Easy on the eyes for late-night coding sessions.
